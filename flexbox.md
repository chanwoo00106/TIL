# flexbox

여러가지 태그를 수평으로 정렬하는데 쓰인다.

## flex-direction

item 들의 정렬과 배치 방향을 설정하는 속성

값|의미
--|--
row|flex의 기본 값으로 왼쪽 정렬이 됨
row-reverse|오른쪽에서 왼쪽으로 정렬이 됨
column|위에서 아래로 정렬이 됨
column-reverse|아래서 위로 정렬이 됨
